# Zlto Concept Pitch

A concept web page and short film built ahead of my interview for the Applied AI Internship at Zlto NPC, exploring how Zlto's platform turns everyday contribution into tangible value and opportunity for young people in South Africa.

## About this project

Rather than only speaking to Zlto's mission in an interview, I wanted to demonstrate applied thinking directly — combining a short promotional film with a one-page site designed around Zlto's core loop: earning tokens for good deeds, spending them on essentials, and building a verified digital record that helps bridge users into employment.

The page's central visual, a live-style "passbook" ledger, was chosen deliberately to reflect how Zlto's platform actually works: contribution and redemption tracked as a running, visible record.

## What's included

- A short film introducing Zlto's model and impact
- A single-page site (`index.html`) built with vanilla HTML/CSS, no external frameworks
- Design language built around Zlto's public mission, not its brand assets

## Disclaimer

This is an independent, unofficial concept project created by a candidate as part of an internship application process. It is not affiliated with, endorsed by, or produced on behalf of Zlto NPC. All colours and layout are original.
